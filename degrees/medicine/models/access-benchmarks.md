# Competitiveness benchmarks for Medicine in Spain

## Objective

This document defines how the system evaluates the competitiveness of Medicine degrees across universities.

It does not contain specific university data, but provides a framework to interpret it.

---

## Core principle

Competitiveness represents how difficult it is to gain admission to a university.

The system must always prioritize:

> Maximizing probability of admission over perceived prestige

---

## Key factors influencing competitiveness

### Public universities

Competitiveness is primarily driven by:

- Cut-off grade (nota de corte)
- Number of available seats
- Number of applicants

General rule:

- Higher cut-off → higher competitiveness
- Lower cut-off → lower competitiveness

---

### Private universities

Competitiveness in private universities is not determined by a single factor, but by the interaction of multiple components.

The system must evaluate all factors together, not in isolation.

---

#### 1. Selection filter

Defines how restrictive the initial access to the admission process is.

- Minimum required academic grade (if any)
- Pre-screening mechanisms
- Candidate filtering before exams or interviews

Two types of filters may apply:

- Absolute filters → minimum grade required to apply
- Competitive filters → only top candidates are invited to continue

A stricter selection filter increases competitiveness.

---

#### 2. Evaluation model

Defines how the final admission score is calculated.

- Weight of entrance exam
- Weight of academic record (Bachillerato)
- Additional components (interviews, internal criteria)

The evaluation model determines which type of student profile is favored by the university.

Implications:

- High exam weight → favors strong test performance
- High academic weight → favors strong prior grades
- Balanced model → rewards consistency across both dimensions

---

#### 3. Competition level

Represents demand relative to supply.

- Number of applicants
- Number of available seats
- Estimated applicants per seat ratio

This factor represents external pressure on admission, independent of the evaluation model.

Higher ratios increase competitiveness.

---

#### 4. Admission flexibility

Defines how many opportunities the student has to secure a place.

- Number of admission rounds
- Rolling vs fixed admissions

Higher flexibility reduces effective competitiveness over time.

This factor interacts strongly with timing and can compensate for other disadvantages.

---

## Competitiveness scale

The system classifies universities into:

- **Very High competitiveness** → extremely difficult to access
- **High competitiveness** → very selective
- **Medium competitiveness** → selective but achievable
- **Low competitiveness** → moderately competitive
- **Very Low competitiveness** → more accessible

---

## Interpretation rules

### Public universities

Competitiveness can be approximated using cut-off grades:

- Cut-off ≥ 13.0 → Very High competitiveness
- Cut-off ~12.7–12.99 → High competitiveness
- Cut-off ~12.3–12.69 → Medium competitiveness
- Cut-off <12.3 → Lower competitiveness

These thresholds are indicative and may evolve over time.

---

### Private universities

Competitiveness must be inferred by combining:

- Selection filter
- Evaluation model
- Competition level
- Admission flexibility

No single factor should be used in isolation.

---

## Public universities - timing factor

Timing affects effective competitiveness.

### July (main allocation)

- Full competition pool
- Highest cut-off levels
- Main admission opportunity

### September (second allocation)

- Limited remaining seats
- Potentially lower cut-off
- Less predictable outcomes

---

## Strategic implications

The system must consider:

- Academic level + competitiveness
- Student strengths (exam vs academic record)
- Timing of the application
- Flexibility of the admission process

A strong strategy typically includes:

- One ambitious option
- One or two realistic options
- One safe option

---

## Usage in recommendation logic

This framework allows the system to:

- Translate raw data into actionable insight
- Avoid static or outdated recommendations
- Adapt to each student profile dynamically

The system must never rely on static classifications alone.
